📁 使用方法

1. 複数ファイル選択: Ctrl+クリック（Windows）またはCmd+クリック（Mac）で複数選択可能
2. 個別名前選択: 各ファイルに対して個別にキャラクター名を選択
3. 保存方法選択:
• 通常ダウンロード: ダウンロードフォルダに保存（重複時は番号が付きます）
• 上書き保存 (推奨): 初回のみ場所を選択、以降は自動上書き（番号が付きません）
4. 配置先: images/changeable/flying-characters または walking-characters
💡 上書き保存の利点:
• WebSocketで頻繁に画像更新する際に最適
• ファイル名(1).jpg、ファイル名(2).jpgのような番号が付かない
• 常に最新のファイルのみが存在
• ダウンロードフォルダが散らからない
